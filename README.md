# Narasimha Group Website Clone

## Overview
A modern, fully responsive 7-page website clone for Narasimha Group - a business group based in Ankola, Karnataka offering grocery, financial services, supplies, and retail stores.

## Project Structure
```
/
├── index.html              # Home page
├── server.py               # Python HTTP server
├── css/
│   └── styles.css          # Main stylesheet with animations
├── js/
│   └── main.js             # JavaScript for animations and interactivity
├── images/
│   └── logo.png            # Company logo
├── pages/
│   ├── big-bazar.html      # Big Bazar page
│   ├── financial-services.html  # Financial Services page
│   ├── suppliers.html      # Suppliers page
│   ├── stores.html         # Stores page
│   ├── about.html          # About page
│   └── privacy-policy.html # Privacy Policy page
└── attached_assets/        # Original reference assets
```

## Design System
- **Primary Color**: #31A050 (Green)
- **Accent Color**: #727586 (Gray)
- **Background Dark**: #1D1E20
- **Text Primary**: #1D1E20
- **Fonts**: Outfit (headings), DM Sans (body), Inter (paragraphs)

## Features
- Fully responsive design (mobile, tablet, desktop)
- Smooth scroll animations
- Mobile hamburger menu with slide-in animation
- Contact forms with validation
- Interest rate tables for Financial Services
- Brand logo grid for Suppliers
- Image galleries

## Running the Website
The website runs on a Python HTTP server on port 5000.
```bash
python server.py
```

## Pages
1. **Home** - Overview of all services with hero section
2. **Big Bazar** - Retail store information with gallery
3. **Financial Services** - Interest rates and schemes
4. **Suppliers** - Brand partners and wholesale info
5. **Stores** - Store location with map
6. **About** - Company information
7. **Privacy Policy** - Privacy and data handling
